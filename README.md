🎬 WebCinema — Система управления кинотеатром
📌 О проекте
WebCinema — это полнофункциональное веб-приложение для автоматизации работы современного кинотеатра. Система разработана в рамках курсового проекта и предоставляет удобный интерфейс для управления фильмами, сеансами, продажей билетов и пользователями с разграничением прав доступа.

⚙️ Функциональные возможности
👤 Гость (без авторизации)
Просмотр списка фильмов с информацией о жанре, длительности и возрастном рейтинге

Просмотр расписания сеансов (предстоящие и прошедшие за месяц)

Детальная информация о каждом сеансе (дата, время начала/окончания, номер зала)

💳 Кассир (после авторизации)
Продажа билетов — визуальная схема зала с отображением свободных и занятых мест, типы мест с разными ценами (Стандарт, VIP, Премиум)

Оформление продажи — указание ФИО зрителя, выбор способа оплаты (Наличные, Карта, QR-код)

Возврат билетов — поиск по сеансу и возврат билетов с обновлением статуса места

Просмотр проданных билетов — фильтрация по сеансу, дате, фильму и залу

Формирование отчета — отчет по сеансу в формате Microsoft Word с подписями и датой

👑 Администратор (после авторизации)
Управление фильмами — добавление, редактирование, удаление

Управление сеансами — добавление, редактирование, удаление с автоматической проверкой на пересечение сеансов в одном зале

Управление тарифами — добавление, редактирование, удаление ценовых категорий

Управление пользователями — добавление, редактирование, удаление кассиров и администраторов

Просмотр архива — история удаленных записей и логи действий пользователей

🗄️ Структура базы данных
Основные таблицы
Таблица	Описание
users	Пользователи системы (логин, пароль, роль, ФИО)
movies	Фильмы (название, длительность, жанр, возрастной рейтинг)
halls	Залы кинотеатра (номер зала)
tariffs	Тарифы (название, цена)
sessions	Сеансы (дата, фильм, время начала/окончания, зал)
seats	Места в залах (ряд, место, тип, привязка к тарифу)
tickets	Билеты (сеанс, место, зритель, цена, способ оплаты, статус)
*_archive	Архивные таблицы для удаленных записей
action_logs	Логи действий пользователей
🚀 Установка и запуск
1. Клонирование репозитория
git clone <repository-url>
cd Cinema_Kursovoy
2. Создание виртуального окружения
python -m venv venv
venv\Scripts\activate          # Windows
source venv/bin/activate       # Linux/Mac
3. Установка зависимостей
pip install -r requirements.txt
4. Создание базы данных PostgreSQL
sql
CREATE DATABASE cinema_db ENCODING 'UTF8';
5. Настройка подключения
В файле database.py укажите пароль от PostgreSQL:
DB_PASSWORD = "ваш_пароль"
6. Запуск приложения
uvicorn main:app --reload --host 127.0.0.1 --port 8000
7. Доступ к приложению
Откройте браузер и перейдите по адресу:
http://127.0.0.1:8000

📄 Генерация отчетов
Для формирования отчета по сеансу:

Войти в систему как кассир

Перейти в раздел «Проданные билеты»

Выбрать сеанс из списка

Нажать кнопку «Сформировать отчет WORD»

Отчет содержит:

Информацию о сеансе (фильм, дата, время, зал, количество билетов)

Таблицу с проданными билетами (ФИО, ряд, место, цена, оплата, статус)

Итоговую сумму

Место для подписи директора

Дату составления отчета
